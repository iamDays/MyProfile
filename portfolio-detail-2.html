Vendor Management System

Vendor Management System

Aplikasi web full-stack untuk mengelola data vendor, kontrak, dan proses pengadaan barang/jasa. Sistem ini membantu tim procurement untuk tracking status vendor, evaluasi performa, dan memastikan compliance terhadap kebijakan perusahaan.

Dibangun dengan React untuk frontend yang responsif dan Node.js untuk backend API. Fitur utama mencakup CRUD vendor, manajemen kontrak dengan reminder otomatis, dan dashboard analitik untuk monitoring spend per kategori.

Fitur Utama

  • Vendor registration & onboarding workflow
  • Contract management dengan expiry alerts
  • Vendor performance scoring & rating system
  • Role-based access control (Admin, Buyer, Viewer)
  • RESTful API dengan dokumentasi Swagger
