Sales Performance Dashboard

Sales Performance Dashboard

Dashboard interaktif yang dirancang untuk memantau dan menganalisis kinerja penjualan secara real-time. Proyek ini melibatkan pengumpulan data dari berbagai sumber, pembersihan data, dan pembuatan visualisasi yang mudah dipahami oleh stakeholder.

Dashboard ini menampilkan KPI utama seperti total revenue, conversion rate, pertumbuhan month-over-month, dan perbandingan performa antar tim sales. Menggunakan chart interaktif yang memungkinkan drill-down ke level detail untuk analisis lebih mendalam.

Fitur Utama

  • Real-time data refresh dari database SQL
  • Interactive bar charts & line graphs untuk trend analysis
  • Filter berdasarkan region, produk, dan periode waktu
  • Export report ke PDF dan Excel
  • Automated email alerts untuk target yang tidak tercapai

Informasi Proyek

  • Kategori Data Analytics
  • Tools Python, SQL, Tableau
